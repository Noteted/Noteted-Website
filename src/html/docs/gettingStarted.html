Gettin' started with Noteted

    Here are the basics you need to know.

  • Noteted has a markdown support. All your markdown files are rendered as HTML and a preview on the right.
  • Noteted has a simplistic choice of UI for ease of use.
  • Noteted has a custom TODO list file support called .td.
  • Noteted is built in Python and has rendering issues with customtkinter, but no one cares anyways.
