Introducing the TODO Renderer

This thing was a pain in the ass to code, but it was worth it.
I'll let the gif speak for itself.

Features

  • [+] Plus, it'll make a new todo.
  • [#] Hashtag, it'll make a new board.
  • [x] X'ed box, it'll mark the todo as... well... completed.
  • [~] Tilde'ed box, it'll mark the todo as... well... cancelled.

  • [+] Plus in a todo, it'll add a new sub-todo.
  • [x] Red X marks the delete! It'll delete the todo/board.

  • Right click, it'll mark the todo as cancelled.
  • Double click on a todo/board, it'll edit the todo/board name.

Few key things to note:

  • The renderer is a BUGGY MESS and has rendering problerms. Don't worry, I'm planning to improve the renderering in the future.
